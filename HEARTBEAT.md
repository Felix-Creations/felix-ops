Run these checks on each heartbeat. Be concise. Only message Shane if something actually needs attention.

## Checks (rotate, don't repeat all every cycle)

- **Email**: Scan for anything urgent or time-sensitive
- **Calendar**: Any events coming up in the next 2 hours?
- **Tasks**: Any queued work that can be progressed autonomously?
- **Projects**: Any running processes or background tasks to report on?

## Rules

- If nothing needs attention → reply HEARTBEAT_OK (no message sent)
- If something needs attention → alert Shane with a short, clear message
- Don't check-in just to check-in — only message when it matters
- Time window: 07:00–24:00 AEST (auto-enforced)
